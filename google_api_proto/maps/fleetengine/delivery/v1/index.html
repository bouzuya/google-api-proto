Task

A task in the Delivery API represents a single action to track. In general there is a distinction between shipment-related tasks and break tasks. A shipment can have multiple tasks associated with it; for example, one task for the pickup and one for the dropoff or transfer. Different tasks for a given shipment can be handled by different vehicles; for example, one vehicle handles the pickup and drives the shipment to the hub, while another drives the same shipment from the hub to the dropoff.
